# kitty-conf

A browser-based conf editor / generator for the [kitty](https://sw.kovidgoyal.net/kitty/) terminal emulator.

Change settings, see a live preview, and copy the generated `kitty.conf`.

**Try it:** [kitty-conf.balintb.com](https://kitty-conf.balintb.com/)

## Development

```sh
bun install
bun run dev
```

## Build

```sh
bun run build
```
